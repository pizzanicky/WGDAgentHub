WGDAgentHub

Agent 可视化工作台

不需要记命令。直接在页面里填内容、上传文件、查看结果。

支持 3 个场景 Jira 工时抓取 / 工时预估 / 数据分析

Jira 工时抓取

输入项目 Key，自动整理 Story 和任务预估，输出 Markdown 报告。

工时预估

可以直接粘贴需求，也可以上传 Markdown 文件。两者都填时，以文本框内容为准。

数据分析

上传 Excel 或 CSV，选择分析模板，可选填写收件邮箱。
